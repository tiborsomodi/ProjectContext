SwitchContext ANT task manual

Generates output based on the given template using the given properties as variables. Uses Velocity template language.

Reference

Attribute Description Required
contextRelativePath Path of context direcory, relative to basedir Yes
templateDir Name of directory which contains templates. Defaults to "template" No
propDir Name of directory which contains properties. Defaults to "property" No
propName Properties file has to located in propDir with name "propName_contextName.props". Default for propName is resourceName No
resourceName Name of the resource Yes
contextName Name of the context Yes
outRelativePath Output's path, relative to basedir Yes

Example usage

Define the task:
<taskdef name="switchcontext" classname="com.inepex.projectcontext.SwitchContextTask">
	<classpath>
		<fileset dir="lib">
			<include name="*.jar" />
		</fileset>
	</classpath>
</taskdef>

"lib" can be change if you use different basedir

<input message="Please enter requested context name:" addproperty="contextName" />
<switchcontext contextRelativePath="ProjectContext/context" resourceName="web.xml" contextName="${contextName}" outRelativePath="ProjectContext/out" />

It will look for templates in basedir/ProjectContext/context/templates and generate output to basedir/ProjectContext/out
